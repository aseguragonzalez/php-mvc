# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting a vulnerability

Please do **not** report security vulnerabilities via GitHub Issues.

Send a description of the vulnerability to **a.segura.gonzalez@gmail.com** with the subject line `[SECURITY] mvc-framework`. Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions

You will receive an acknowledgement within 48 hours. Once the issue is confirmed, a patch will be prepared and a new release published before any public disclosure.
